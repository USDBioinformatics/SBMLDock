
1. Put Modeller jar in any folder,

2. Run Jar via CMD through following command
	java -jar SBMLModeler.jar <Path to store file> <Pathway name>
	
	e.g java -jar SBMLModeler.jar "F:/Files" "folate biosynthesis"
	
Note: Pathways must be selected from Pathway List files provided in same folder and Destination Path should not contain trailing "/" after folder name
